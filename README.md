# MyBnB
Book and Host Accommodation

after clone the repository : 
1- create .env in server folder and you should have :
PORT=
DB_USER=
DB_USER_PASSWORD=
DB_CLUSTER=
G_CLIENT_ID= " this for google geolocator Api ID"
G_CLIENT_SECRET=
PUBLIC_URL=
SECRET=
NODE_ENV=
G_GEOCODE_KEY=
S_SECRET_KEY=
CLOUDINARY_KEY=
CLOUDINARY_SECRET=
CLOUDINARY_NAME=


2 - run " npm i " 

3- run " npm start " and start use the app
